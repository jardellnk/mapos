Antes de instalar o Maostore e fazer qualquer modificação, é crucial realizar um backup. Não posso me responsabilizar por conflitos que possam surgir durante o processo.

Inicie configurando o emitente e o sistema. Se a logo não estiver atualizada, isso pode resultar em erros. Recomenda-se editar as informações do emitente antes de prosseguir com os scripts, já que não há um campo designado no Mapos para inserir esses detalhes.

Após a configuração inicial, é necessário atualizar as tabelas do MySQL utilizando o script fornecido.

É importante notar que, por padrão, os produtos não serão exibidos na vitrine. Para autorizar sua exibição, é necessário modificar a coluna "exibir" do produto desejado. Quando o valor for igual a 0, os produtos estarão ocultos. Para exibi-los, altere o valor para 1.

Certifique-se de seguir essas etapas com atenção para garantir uma instalação adequada e evitar possíveis problemas futuros.

CONFIGURAÇOES EXTRA ------>
para mudar a pagina principal

editar o config.php 
\mapos\application\config\config.php

$config['csrf_exclude_uris'] = ['api.*+'];
$config['csrf_exclude_uris'] = ['api.*+','home.*+'];

editar o router 
\mapos\application\config\routes.php

$route['default_controller'] = 'mapos';
$route['404_override'] = '';

$route['default_controller'] = 'home';
$route['mapos'] = 'mapos';
$route['404_override'] = '';

EXTRA DEIXANDO URL AMIGAVEL 

crie um arquivo na pasta raiz do mapos
.htaccess

RewriteEngine On
RewriteBase /mapos/ <----- nome da pasta onde contem os arquivos do mapos -- caso esteja em hospedagem web e o mapos esteja na raiz ou em subdominio so deixar "/" uma barra

# Exceção para /index.php/api/v1
RewriteCond %{REQUEST_URI} ^/index\.php/api/v1 [NC]
RewriteRule ^ - [L]

# Redirecionar todas as URLs que contêm /index.php/ para URLs sem esse trecho
RewriteCond %{THE_REQUEST} \s/index\.php/([^\s?]+) [NC]
RewriteRule ^ %1 [L,R=301]

# Roteamento interno para index.php
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^ index.php [L]


edite o config.php

$config['index_page'] = 'index.php';
$config['index_page'] = '';
